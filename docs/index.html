How to Use

Google Calendar

  1. Click the "Download Calendar" button above
  2. In Google Calendar, click the "+" next to "Other calendars"
  3. Choose "From URL"
  4. Paste this URL: https://defeomike.github.io/strava-club-ride-calendar/calendar.ics
  5. Click "Add calendar"

Apple Calendar

  1. Click the "Download Calendar" button above
  2. Open the downloaded .ics file
  3. Click "Subscribe" in the prompt

Other Calendar Apps

Most calendar applications support .ics files. You can either:

  • Download and open the .ics file directly
  • Subscribe to the calendar URL: https://defeomike.github.io/strava-club-ride-calendar/calendar.ics

About

This calendar is automatically updated with events from Strava cycling clubs. Events are set to notify 30 minutes before the start time.
